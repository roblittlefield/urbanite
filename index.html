Welcome to Urbanite! 🌁

    Explore the latest law enforcement dispatched calls for service in San Francisco with this interactive web app. Move around the map to view calls in different neighborhoods. Change the map skin. Open the list to see the latest calls in all SF or nearby.

    Built as a lightweight and speedy alternative to other heavier incident mapping apps with pay-walls and ads.

    A personal project designed and developed in the Misson using vanilla Javascript. 🌯🌴🌴

    Inquiries/feedback? DM @roblittlefield

    Calls anonymized to nearest intersection, delayed 10m. Links to source details in footer.
